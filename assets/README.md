Optional local media files for the MVP website.

If you want the website to load your rendered clips automatically, place them in
this folder with these filenames:

- `dispatch.mp4` for the dispatch/setup scene
- `escalation.mp4` for the on-scene escalation clip
- `immediate.mp4` for Outcome A (rushed extraction)
- `secure.mp4` for Outcome B (scene stabilized first)

If these files are missing, the site still works using styled story frames.
You can also upload clips directly from the Media Workbench in the browser.
